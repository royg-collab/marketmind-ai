# 📊 MarketMind AI – GenAI-Powered Stock Insight Assistant

AI-powered stock insight assistant that helps retail investors understand market trends using simple explanations instead of complex data.

---

## 📸 Demo
![App Screenshot](demo.png)


---

## 🚨 Problem
Retail investors struggle to interpret stock data due to:

- Complex charts  
- Financial jargon  
- Misinformation  

---

## 💡 Solution
MarketMind AI:

- Fetches real-time stock data  
- Calculates key indicators (MA50, MA200)  
- Generates BUY/SELL signals  
- Uses Generative AI to explain decisions in simple language  

---

## 🧠 Why GenAI?
Instead of just showing signals, the system uses Generative AI to explain *why* a decision is made, making stock analysis easy even for beginners.

---

## ⚙️ Tech Stack
- Python  
- Streamlit  
- yfinance  
- OpenAI API  

---

## ▶️ How to Run

### 1. Clone the repository
```bash
git clone https://github.com/royg-collab/marketmind-ai.git
cd marketmind-ai
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
streamlit run app.py
```
